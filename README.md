# Stock-Predictor
# Stock Price Prediction using Machine Learning

## Overview

This project predicts stock price trends using historical data and machine learning techniques.

## Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn / TensorFlow
* Matplotlib

## Features

* Real-time stock data using yfinance
* Data preprocessing and scaling
* Trend prediction model
* Visualization of predicted vs actual prices

## Results

The model successfully captures trends in stock price movement and demonstrates the potential of ML in financial forecasting.

## Future Improvements

* LSTM-based deep learning model
* Multi-stock prediction
* Deployment as a web app
